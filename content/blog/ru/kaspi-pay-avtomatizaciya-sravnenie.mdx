## AiPay

**Что это:** платформа для автоматизации Kaspi Pay платежей. Вы создаёте счета через API или Telegram-бот, клиент платит через Kaspi, вы получаете автоматическое подтверждение.

**Ключевые возможности:**
- REST API для интеграции с вашей CRM, сайтом или Telegram-ботом
- Webhook — мгновенные уведомления при смене статуса оплаты
- Sandbox — тестируйте интеграцию без реальных денег
- Поддержка на русском, казахском и английском
- Интеграция с AmoCRM, Bitrix24, Telegram, WhatsApp

**Цены:** от $49/мес за базовый тариф. Бесплатный trial на 14 дней.

**Плюсы:**
- Нативная интеграция с Kaspi Pay
- Полная автоматизация — от создания счёта до подтверждения оплаты
- REST API и webhook для разработчиков
- Поддержка на WhatsApp и Telegram
- Казахстанская компания — понимание локальных требований

**Минусы:**
- Работает только с Kaspi Pay (не с Halyk, Jusan и др.)
- Требует POS-терминал Kaspi (получается бесплатно)

**Кому подходит:** бизнесам в Казахстане, которые принимают Kaspi Pay и хотят автоматизировать учёт, верификацию и отчётность.

## Apipay

**Что это:** китайская платёжная система для跨境 (кросс-бордер) платежей. Позволяет принимать платежи от китайских покупателей через Alipay, WeChat Pay и другие китайские платёжные методы.

**Ключевые возможности:**
- Интеграция с Alipay, WeChat Pay, UnionPay
- Кросс-бордер платежи (Китай → остальной мир)
- API для онлайн-магазинов
- Поддержка 100+ стран

**Цены:** комиссия 2-3% за транзакцию + ежемесячный сбор.

**Плюсы:**
- Работает с китайскими платёжными системами
- Подходит для e-commerce с аудиторией из Китая
- Поддержка множества валют

**Минусы:**
- **Не работает с Kaspi Pay** — это главная причина, почему Apipay не подходит для казахстанского бизнеса
- Комиссии выше, чем у локальных решений
- Поддержка в основном на китайском и английском
- Не адаптирован под казахстанское законодательство

**Кому подходит:** онлайн-магазинам, которые продают товары китайским покупателям или импортируют из Китая.

## PayBot

**Что это:** Telegram-бот для приёма платежей. Создаёте бота, настраиваете каталог товаров, клиенты оплачивают через бота.

**Ключевые возможности:**
- Приём платежей через Telegram
- Каталог товаров
- Простая настройка без кода
- Интеграция с Kaspi Pay (базовая)

**Цены:** от $29/мес.

**Плюсы:**
- Простая настройка — без разработчика
- Подходит для микробизнеса
- Работает внутри Telegram

**Минусы:**
- Ограниченный функционал — нет REST API, нет webhook
- Не масштабируется — подходит только для малого объёма транзакций
- Нет sandbox для тестирования
- Поддержка только в Telegram
- Нет интеграции с CRM (AmoCRM, Bitrix24)

**Кому подходит:** микробизнесу, который продаёт через Telegram и не нуждается в автоматизации.
